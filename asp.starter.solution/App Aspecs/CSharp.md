## C# naming conversion

| Item | Convention | Example |
| :--- | :--- | :--- |
| **Class** | **PascalCase** | `public class UserProfile` |
| **Method** | **PascalCase** | `public void CalculateTotal()` |
| **Interface** | **IPascalCase** | `public interface IRepository` |
| **Local Variable** | **camelCase** | `var itemCount = 0;` |
| **Method Parameter** | **camelCase** | `public void Save(string userName)` |
| **Private Field** | **_camelCase** | `private readonly string _connectionString;` |
| **Property** | **PascalCase** | `public int Age { get; set; }` |
